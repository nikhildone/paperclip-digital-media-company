# Paperclip Dashboard Study

Important dashboard areas already studied:

- Dashboard
- Inbox
- Tasks
- Routines
- Goals
- Artifacts
- Projects
- Agents
- Org
- Skills
- Costs
- Activity
- Settings

Useful built-in concepts:

- Agents can own work.
- Tasks and issues represent work items.
- Subtasks can form execution chains.
- Skills can define reusable agent behavior.
- Routines can create recurring work.
- Artifacts can store user-facing outputs.
- Projects separate production and repair work.
- Activity and costs provide monitoring.
- Settings and adapters control execution options.

Current mapping:

- SINK DINK Media Company = production work.
- Onboarding = testing and repair.
- CEO agent = main orchestration agent after run stability is confirmed.
